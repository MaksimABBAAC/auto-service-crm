# STO
Менеджер заказов в автосервисе PHP и PgSQl с аутенфикацией пользователя
***

## Описание проекта
Приложение позволяет аутентифицированному пользователю создать запись.
Используемые технологии : СУБД PgSQL, CSS-феймворк BootStrap
